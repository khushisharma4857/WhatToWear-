# WhatToWear - Feature Prioritization Matrix

## Prioritization Framework: RICE Scoring

**RICE = (Reach × Impact × Confidence) / Effort**

- **Reach**: How many users benefit? (0-10)
- **Impact**: How much does it change user experience? (0-4x multiplier)
- **Confidence**: How sure are we it works? (0-100%)
- **Effort**: Engineering months to build (1-20)

## User Story Examples

### P0: Image Upload & Classification
**As a** new user,
**I want to** photograph my clothes and have them automatically categorized,
**So that** I don't have to manually tag each item.

**Acceptance Criteria**:
- [ ] Camera app opens from app
- [ ] Photo captures clothing item clearly
- [ ] AI returns category within 10 seconds
- [ ] Top 3 attributes detected with > 85% confidence
- [ ] User can edit detected attributes
- [ ] Item saved to wardrobe

---

### P0: AI Styling Assistant
**As a** busy professional,
**I want to** ask "what should I wear today?",
**So that** I can get dressed quickly without thinking.

**Acceptance Criteria**:
- [ ] Chat interface accepts natural language
- [ ] AI returns 3-5 outfit suggestions
- [ ] Each outfit shows full image composition
- [ ] Rationale provided for each suggestion
- [ ] User can regenerate with refinements
- [ ] Recommendations consider wardrobe items only

---

### P0: Outfit Builder
**As a** creative user,
**I want to** visually create outfits by dragging items onto a canvas,
**So that** I can see combinations before wearing them.

**Acceptance Criteria**:
- [ ] Drag-and-drop interface is responsive
- [ ] Items layer correctly (base → top → outerwear)
- [ ] Real-time preview shows full outfit
- [ ] Save outfit with custom name
- [ ] Outfits persist in database
- [ ] Can edit saved outfits

---

### P1: Social Sharing
**As a** fashion enthusiast,
**I want to** share my outfits on social media,
**So that** I can get feedback and inspire my friends.

**Acceptance Criteria**:
- [ ] Share button exports high-quality image
- [ ] Works with Instagram, TikTok, Twitter
- [ ] Image includes outfit details (optional)
- [ ] Shareable link tracks engagement
- [ ] Can make outfits public/private

---

### P1: Weather Integration
**As a** daily user,
**I want** outfit recommendations to consider current weather,
**So that** I'm dressed appropriately for conditions.

**Acceptance Criteria**:
- [ ] Fetches current weather for user location
- [ ] Suggests seasonal items only
- [ ] Filters cold-weather vs. warm-weather items
- [ ] Works without explicit weather selection
- [ ] User can manually override location

---
